Toast

Toast display a temporary notification related to an interaction between user and the website. They usually stay on the screen for 2-3 seconds and then disappear. They're meant to be noticed without disrupting a user's experience or flow. In order to only use the Toast Component from Blaze UI in your project, copy and paste the below code in the head tag of your html document. 

                            
            <link rel="stylesheet" href="https://blaze-ui.netlify.app/Components/alert/alert.css" />
            <link rel="stylesheet" href="https://blaze-ui.netlify.app/Components/toast/toast.css" />
