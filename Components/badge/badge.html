Badge on Avatars

Badges indicate the status of the user when used on avatars. Example-> Green: online | Red: offline. For using badge on avatars, you can add the class name badge to the parent tag along with the variation class on child, for instance: badge-online | badge-offline | badge-dnd. 

                        
                            <div class="badge avatar avatar-lg-size">
                                <img class="avatar-round" src="/assets/avatar-male1.png">
                                <span class="badge-online"></span>
                            </div>
                            <div class="badge avatar avatar-lg-size">
                                <img class="avatar-round" src="/assets/avatar-fm1.png">
                                <span class="badge-offline"></span>
                            </div>
                            <div class=" badge avatar avatar-lg-size">
                                <img class="avatar-round" src="/assets/avatar-fm2.png">
                                <span class="badge-dnd"></span>
                            </div>

Badge on Icons

Badges indicate the notification count on a particular item. For using badge on icons, you can add the class name badge to the parent tag along with the badge-number class name on child.

                                <i class="fas fa-shopping-cart"></i>
                                <div class="badge-number badge-offline semibold">4</div>
